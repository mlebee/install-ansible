# Install Ansible

This script is meant for quick & easy setup Ansible and its requirements below:
- apt packages:
```
  redis-server
  python-mysqldb
  python-apt
  python-netaddr
```

- pip packages:
```
  dnspython
  docker-py
  pyvmomi
  redis
```


### HowTo

Run this curl comand as root:
```
curl -sSL https://raw.githubusercontent.com/mlebee/install-ansible/master/setup.sh | sudo bash
```
